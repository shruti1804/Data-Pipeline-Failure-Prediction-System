# 🚀 Data Pipeline Failure Prediction System

## 📌 Overview

This project aims to build a system that predicts potential failures in data pipelines using machine learning.
It is currently under development.

---

## 🎯 Goal

To create an end-to-end system that:

* Monitors pipeline logs
* Predicts failures in advance
* Provides insights through a dashboard

---

## 🛠️ Tech Stack (Planned)

* Python (Flask)
* PostgreSQL
* Scikit-learn
* HTML, CSS, JavaScript (Chart.js)
